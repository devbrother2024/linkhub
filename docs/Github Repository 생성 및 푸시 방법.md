## Github Repository 생성 및 현재 프로젝트 푸시

- Github CLI (GH) 설치가 완료되어 있어야 함
- Github 로그인 & git 프로젝트 초기화

```sh
gh auth login
git init
git config user.name devbrother2024
git config user.email devbrother2024@gmail.com
```

- Github Repository 생성 및 main 브랜치 푸시

```text
Github Repository "원하는레포이름" 생성해서
현재 모든 로컬 변경사항 커밋 후 main 브랜치에 푸시해줘.
```
